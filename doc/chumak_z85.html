Function Details

decode/1

decode(String::string()) -> binary()

Decode a Z85 string to a binary.

As per the spec, the length of the string SHALL be divisible by 5 with no remainder.

Note that not all strings are valid encodings. The result for a string that is not a valid encoding is unspecified.

encode/1

encode(Binary::binary()) -> string()

Encode a binary to a Z85 string.

As per the spec, the length of the binary SHALL be divisible by 4 with no remainder.
