This module interfaces to the encryption library.

Description

This module interfaces to the encryption library.

Three variants are supported: nacl, enacl and nacerl. Which variant is used is determined at the time of compilation.

If no encryption library is available, an error is thrown.

Encrypts+authenticates a message to another party.

Encrypt a Message to the party identified by that party's public key using your own secret key to authenticate yourself. Requires a Nonce in addition. Returns the encrypted message.
